Présentation

3en1Compteur est un compteur de visites écrit en PHP. Il est simple et sans fioriture. La mise en place se fait en quelques minutes seulement !

Fonctionnalités de base :

  • Affichage du nombre total de visites
  • Affichage du nombre de visites quotidiennes
  • Affichage du nombre de visiteurs en ligne

Autres fonctionnalités :

  • Possibilité de définir un nombre de visiteurs initiaux
  • Exclusion d'IPs partielles ou totales
  • Conservation des IPs pendant une période définie par l'utilisateur
  • Définition d'un intervalle de temps pour le nombre de visiteurs en ligne
  • Définition de la période durant laquelle un visiteur est considéré comme unique

Prérequis fonctionnels

  • PHP version 5.X.X
  • MySQL versions 4.1.13 ou plus récent, ou 5.0.7 ou plus récent

Installation

  1. Ouvrez le fichier config.inc.php dans un éditeur de texte. Les paramètres du compteur peuvent être laissés à leurs valeurs par défaut. Sauvegardez le fichier puis fermez-le.
    • Renseignez tous les champs des paramètres de connexion.
    • Les paramètres du compteur peuvent être laissés à leurs valeurs par défaut.
    • Sauvegardez le fichier puis fermez-le.
  2. Renommez le dossier 'upload' comme il vous plait, par exemple 'compteur'. Uploadez-le, avec tout ce qu'il contient, quelque part sur votre serveur.
  3. Depuis votre navigateur, tapez l'adresse pointant vers le fichier install.php du compteur.
    Exemple : http://www.votresite.com/compteur/install.php.
  4. L'installation réussie, copiez le code fourni et collez-le sur toute page utilisant le compteur. Si vous perdez ce code, uploadez le fichier codes.php et pointez votre navigateur vers lui de la même façon que précédemment.
  5. Supprimez le fichier install.php, c'est tout !

FAQ

Je souhaite récupérer le code à insérer dans mes pages, comment faire ?

Dans l'archive, vous trouverez un fichier codes.php. Uploadez ce fichier dans le répertoire du compteur puis pointez votre navigateur vers lui.

J'ai réinstallé le compteur, mais je ne veux pas réinitialiser le total de mes visites, comment faire ?

Dans le fichier config.inc.php changez la valeur du paramètre $initial.

Comment connaître les IPs à bloquer ? Et comment les bloquer ?

En vous rendant dans la BDD vous pouvez voir sur la colonne c_host les hosts correspondants à chaque IP. Vous verrez par exemple que les IPs commençant par 66.249. correspondent au GoogleBot.
Vous pouvez alors décider de ne pas les comptabiliser en les ajoutant au paramètre $exclude dans le fichier config.inc.php.

Comment puis-je connaître la configuration de mon serveur ?

Créez une page phpinfo.php vers laquelle vous pointerez et insérez-y ce code :

<?php phpinfo(); ?>

Chez OVH, PHP 5 est installé mais c'est PHP 4 qui est activé par défaut. Que faire ?

Dans un fichier .htaccess à la racine de votre site ajoutez cette ligne :

SetEnv PHP_VER 5
